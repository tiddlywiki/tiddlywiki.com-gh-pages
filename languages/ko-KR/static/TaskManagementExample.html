TaskManagementExample

2018년 3월 7일 오후 3:35

TiddlyWiki5 can be used as a simple task management system without further customisation.
The idea is that tasks be tagged task, with those that are completed also tagged done. In this way it is straightforward to generate task lists.

Tip
There is an enhanced version of this demo that adds the ability to drag and drop the tasks to re-order them.

Outstanding tasks

<$list filter="[!has[draft.of]tag[task]!tag[done]sort[created]]">
<$checkbox tag="done"> <$link/></$checkbox><br>
</$list>
