Release 5.4.0

Released TBA

See GitHub for detailed change history of this release

Introduction

Release v5.4.0 includes major improvements in a number of areas. See the project plan for more details.

Changes

This release includes the following changes.

Internal

  • PLUGINISATION Offload server-only components to a plugin

    This change reduces the size of the core plugin by 114.5KB or about 4.5%.

  • ENHANCEMENT Split release notes into individual change notes

    Doing so enables us to filter and group changes. For example, we could show all the breaking changes between two releases.
