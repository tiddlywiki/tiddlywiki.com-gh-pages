Imported tiddler definition

Using the data widget to create a tiddler that is a copy of the tiddler "Hello" with the addition of the field "custom" set to "Alpha"
titleOutput
[
    {
        "title": "Hello",
        "modifier": "JoeBloggs",
        "text": "This is the Hello tiddler",
        "custom": "Alpha"
    }
]
