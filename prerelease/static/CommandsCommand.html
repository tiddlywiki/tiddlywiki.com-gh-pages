Commands

Sequentially run the command tokens returned from a filter

--commands <filter>

Examples

--commands "[enlist{$:/build-commands-as-text}]"
--commands "[{$:/build-commands-as-json}jsonindexes[]] :map[{$:/build-commands-as-json}jsonget<currentTiddler>]"
