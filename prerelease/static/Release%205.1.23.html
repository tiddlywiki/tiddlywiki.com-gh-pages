22nd April 2020 at 5:32pm
ReleaseNotes

See GitHub for detailed change history of this release

Major Improvements

Translation Improvements

Plugin Improvements

  • Freelinks Plugin
    • Fixed bug with autolinking within HTML <a> elements
  • Twitter Plugin
    • Added warning if wiki needs to be saved and reloaded

Performance Improvements

Usability Improvements

  • Updated Vanilla theme to use palette colours for the browser selection outline
  • Added warning message about using the online plugin library with the client-server configuration
  • Improved Gruvbox palette readability of toolbar buttons

Hackability Improvements

  • Updated LinkWidget to work within SVG elements
  • Added accept attribute to the BrowseWidget
  • Extended ActionPopupWidget to allow popups to be dismissed
  • Extended favicon mechanism to support _canonical_uri images

Bug Fixes

  • Fixed restored missing parameter to saveTiddler() method sync adaptors
  • Fixed MakeLibraryCommand to skip non-directories
