Warning
It is not recommended to use these classes for styling. Though tiddlywiki might still support them, they may be in the process of being dropped, or may only be kept for compatibility purposes. Avoid using them, and update existing code if possible. Be aware that this feature may cease to work at any time.

These Core Classes are considered deprecated. It is not recommend to use them for styling.

  • tc-tagged-* Deprecated fromv5.1.16 Use Custom styles by data-tags instead.
  • tc-reveal Deprecated fromv5.3.8 for styling purposes as it is subject to change.
  • tc-language-(language code) Deprecated fromv5.3.8 Please use :lang() instead.
